### Aliases Semânticos
| Token | Valor (Alias) | Uso |
| :--- | :--- | :--- |
| `--color-primary` | `blue-500` | Botões primários, links ativos |
| `--color-danger` | `red-500` | Botões destrutivos, erros |
| `--color-success` | `green-500` | Confirmações e sucessos |
| `--color-warning` | `orange-500` | Avisos e estados de atenção |
| `--color-text-primary` | `gray-800` | Texto principal do corpo |
| `--color-bg-primary` | `white` | Fundo principal de páginas |

---

## Acessibilidade (WCAG 2.1 AA)

As combinações de cores devem garantir um contraste mínimo para legibilidade:
- **Texto normal:** Mínimo de 4.5:1
- **Texto grande / Elementos UI:** Mínimo de 3:1

### Exemplos de Contraste Validado
| Combinação (Texto / Fundo) | Contraste | Status |
| :--- | :--- | :--- |
| Gray 800 (`#424242`) sobre White (`#FFFFFF`) | 10.1:1 | ✅ Passa (AAA) |
| Blue 500 (`#005EB8`) sobre White (`#FFFFFF`) | 4.81:1 | ✅ Passa (AA) |
| White (`#FFFFFF`) sobre Blue 700 (`#004587`) | 8.8:1 | ✅ Passa (AAA) |
| Red 700 (`#B00020`) sobre White (`#FFFFFF`) | 7.7:1 | ✅ Passa (AAA) |
| Gray 400 (`#BDBDBD`) sobre White (`#FFFFFF`) | 1.8:1 | ❌ Falha (Não usar para texto) |
