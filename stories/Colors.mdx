### Aliases Semânticos

| Token | Valor | Uso |
| :--- | :--- | :--- |
| `--color-primary` | Azul 500 (#4A82CC) | Botões primários, links |
| `--color-accent` | Teal 500 (#12A08A) | Botões accent, badges especiais |
| `--color-danger` | Danger 500 (#DC2626) | Botões destrutivos, erros |
| `--color-success` | Success 500 (#2E7D32) | Confirmações e sucessos |
| `--color-warning` | Warning 500 (#D97706) | Avisos e estado de atenção |
| `--color-text-primary` | Gray 800 (#252A3A) | Texto principal do corpo |
| `--color-bg-page` | Gray 50 (#F5F7FA) | Fundo de página padrão |
| `--color-bg-navbar` | Navy (#1A2D5A) | Barra de navegação |

## Acessibilidade (WCAG 2.1 AA)

As combinações de cores garantem contraste mínimo para legibilidade:
- **Texto normal:** Mínimo de 4.5:1
- **Texto grande / Elementos UI:** Mínimo de 3:1

### Exemplos de Contraste Validado
