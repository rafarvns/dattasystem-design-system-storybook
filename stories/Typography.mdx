# Tipografia

A tipografia do Datta System foi projetada para equilibrar modernidade e legibilidade funcional, especialmente em resoluções de tela comuns em ambientes governamentais (como 1366×768).

## Famílias Tipográficas

| Token | Família | Uso |
| :--- | :--- | :--- |
| `--font-family-primary` | Inter, Source Sans Pro | Interface do usuário, corpo de texto, formulários e botões. |
| `--font-family-secondary` | Manrope, IBM Plex Sans | Títulos de página e elementos de destaque. |
| `--font-family-mono` | IBM Plex Mono | Dados técnicos, códigos e protocolos. |

## Escala de Tamanhos

A escala utiliza `rem` (base de 16px) para garantir acessibilidade e escalabilidade.

## Pesos e Estilos

| Token | Valor | Uso |
| :--- | :--- | :--- |
| `--font-weight-regular` | 400 | Texto padrão do corpo. |
| `--font-weight-medium` | 500 | Labels e ênfase leve. |
| `--font-weight-semibold` | 600 | Subtítulos e botões. |
| `--font-weight-bold` | 700 | Títulos principais. |

## Presets Tipográficos (Classes)

Estes presets combinam família, tamanho, peso e altura de linha para garantir consistência visual.
