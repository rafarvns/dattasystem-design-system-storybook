# Alert (Alertas e Feedback)

O componente **Alert** fornece feedback visual sobre o estado do sistema ou o resultado de uma ação do usuário. Ele é essencial para comunicar mensagens importantes de forma clara e acessível.

## Quando usar
- Para confirmar o sucesso de uma operação (ex: "Cadastro salvo").
- Para avisar sobre possíveis problemas (ex: "Sessão expirando").
- Para comunicar erros impeditivos (ex: "Falha na conexão").
- Para fornecer informações contextuais relevantes (ex: "Manutenção programada").

## Acessibilidade
- **Roles:** Utiliza `role="alert"` para erros e avisos, e `role="status"` para sucessos e informações.
- **Live Regions:** Configurado com `aria-live="assertive"` para erros e `aria-live="polite"` para os demais.
- **Teclado:** O botão de fechar é focável e pode ser ativado via `Enter` ou `Space`. O alerta também responde à tecla `Esc` quando focado.
- **Contraste:** Todas as variantes foram validadas para atender ao padrão WCAG AA.
