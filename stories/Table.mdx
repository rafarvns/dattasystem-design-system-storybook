# Table (Tabela de Dados)

O componente Table é utilizado para organizar e exibir dados de forma tabular, permitindo que o usuário visualize, compare e interaja com grandes volumes de informação.

## Quando usar
- Para exibir listas de registros (processos, usuários, logs).
- Quando a comparação entre diferentes atributos de um item é necessária.
- Em relatórios e dashboards.

## Anatomia
A tabela é composta por um wrapper (para scroll horizontal), caption descritivo, cabeçalho fixo ou estático, corpo com zebra striping e coluna opcional de ações.

## Estados e Variantes

### Ordenação
Colunas marcadas como `sortable` exibem botões de ordenação que alternam entre ascendente, descendente e nenhum.

<Canvas of={TableStories.StickyHeader} />

### Estado Vazio

<Canvas of={TableStories.Empty} />

## Acessibilidade
- Utiliza tags semânticas nativas (`<table>`, `<thead>`, `<tbody>`, `<th>`, `<td>`).
- `<caption>` obrigatório para descrição da tabela.
- `scope="col"` aplicado em todos os cabeçalhos.
- Gerenciamento dinâmico de `aria-sort`.
- Navegação por teclado via `Tab` entre elementos interativos.
